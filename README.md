# AI Student Performance Prediction

## 🔍 Project Overview

This repository contains an AI‑based solution to **predict student academic outcomes** (pass/fail) using machine learning.  
The project implements and compares several ML models, evaluates their performance, and delivers a real‑time prediction interface using **Streamlit**.

---

## 🧠 Problem Statement

In educational settings, early identification of at‑risk students is critical for timely support and intervention.  
This project builds predictive models based on student academic, behavioral, and socio‑economic features to classify students as likely to pass or fail.

---

## 🚀 Key Features

- Data preprocessing and feature engineering
- Model training and evaluation using:
  - Random Forest
  - Logistic Regression
  - Support Vector Machine (SVM)
  - XGBoost
- Performance metrics (accuracy, confusion matrix)
- Interactive web app built with **Streamlit**
- Model serialization for deployment

---

## 📊 Results Summary
Model  ---  	Accuracy
Random Forest  ---  	89%
Logistic Regression  ---  	84%
SVM  ---  	85%
XGBoost  ---  	88%

The Random Forest model performed best on the test dataset with the highest accuracy and interpretability.
